# Planty

Planty ist ein IoT-Pflanzensensor-Projekt mit Backend, Firmware und späterem Frontend. Ziel ist es, Feuchtigkeitswerte einer Pflanze zu erfassen und daraus Benachrichtigungen auszulösen.

## Repository-Struktur

- `backend/` - Python-Backend und lokale Sensor-/API-Logik
- `frontend/` - geplante Web-Oberfläche
- `firmware/` - Firmware für den D1 Mini / ESP-basierte Sensoren
- `docs/` - Projektdokumentation und Produktbeschreibung

## Status

- Backend-Prototyp in Arbeit
- Firmware-Prototyp vorhanden
- Frontend noch nicht umgesetzt
